# Оформление

## Цветовый схемы
Наименование цветовых [схем](https://squidfunk.github.io/mkdocs-material/setup/changing-the-colors/)

Переключение цветовых схем задается в файле mkdocs.yaml. Схема указанная первой будет схемой по умолчанию.
